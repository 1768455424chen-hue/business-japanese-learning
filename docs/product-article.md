# Business Japanese Skill：一个 Local-first 的 AI 商务日语学习工具

## 摘要

Business Japanese Skill 是一个面向中文使用者的商务日语学习工具。用户可以输入商务日语句子，AI 会自动拆解出词汇、表达、语法、固定搭配等学习点，用户选择想学的内容加入个人学习本，之后通过复习和测试逐步掌握。所有学习数据保存在浏览器本地，用户可以自行配置 AI API Key 来使用真实 AI 解析。

这是一个 Local-first 的个人学习工具，不依赖后端服务，不需要登录账号。

## 一、开发背景

学习商务日语的过程中，真正的难点往往不是单个单词，而是一句话里同时出现的敬语、固定搭配、语法结构和商务专用的表达方式。很多学习者能够读懂一句话的大意，却很难判断其中哪些表达值得单独积累。

使用普通词典查词，只能看到单词本身的释义，无法根据整句的语境来理解表达方式。而 AI 虽然可以帮助分析句子，但如果分析结果看完就关掉，不能沉淀为个人知识库，很快就忘掉了。

这个工具的出发点，就是把 AI 解析的结果变成一个可以复习、可以测试的学习闭环：从句子拆解出学习点，加入个人学习本，通过间隔复习和多种题型测试，最终真正掌握。

## 二、工具的核心功能

### 1. AI 解析：从一句商务日语中拆出学习点

用户可以在 AI 解析页面输入商务日语句子，AI 会自动分析并拆分出多个学习点，每个学习点包含：

- 日语表达和读音
- 中文释义
- 词性
- 例句和中文翻译
- 语法说明
- 常见搭配
- 商务场景说明
- 类似表达

用户可以在分析结果中选择真正想要学习的内容，加入学习本。工具会过滤掉信息不完整的条目，确保学习本中的数据质量。

【请在这里插入图片：03-ai-analyze-result.png】

点击任意学习点可以查看详细说明，包括类似表达、组成词、商务场景说明等。如果类似表达的信息完整，也可以进一步查看并加入学习本。

【请在这里插入图片：04-ai-analyze-detail.png】

### 2. 学習本：把 AI 结果变成自己的知识库

用户选中的学习点会进入学习本（Notebook）。学习本不是简单的收藏夹，而是复习和测试的数据来源。用户可以在学习本中查看所有学习项，点击查看详情，也可以删除不需要的内容。所有数据保存在浏览器本地。

【请在这里插入图片：05-notebook.png】

【请在这里插入图片：06-notebook-detail.png】

### 3. 復習：根据掌握程度持续记忆

学习项会进入复习流程，用户可以根据记忆情况标记「覚えた」或「忘れた」。系统会根据复习结果调整掌握状态和下次复习时间，让学习点从初步接触逐步变成真正能用的知识。

【请在这里插入图片：07-review.png】

### 4. AI テスト：用三种题型确认是否掌握

当学习项达到熟练状态后，可以进入 AI テスト。当前支持三种题型：

- **選択問題**：从四个选项中选择单词的正确释义
- **穴埋め問題**：根据中文释义填写对应的日语单词
- **判断問題**：判断单词在例句中的用法是否正确

每种测试入口会显示当前可测试的対象数量。一项学习内容只有在三种测试中都答对之后，才会被视为完全掌握。対象数量为 0 时不可进入，避免空测试。

### 5. 設定：用户自带 AI API Key

工具默认提供 Mock 模式，可用于体验所有功能。如果要使用真实 AI 解析，用户需要在設定中选择 OpenAI 互換，并填写自己的 API Key、Base URL 和 Model。工具不提供公共 API Key。

推荐配置示例：

- Base URL: `https://api.deepseek.com`
- Model: `deepseek-chat`

其他 OpenAI-compatible provider 也可能可用，但兼容性取决于 provider 是否支持浏览器直连和稳定的 JSON 输出。

## 三、基本使用方法

1. 打开工具，进入 AI 解析页面。
2. 输入在学习或工作中遇到的商务日语句子。
3. 查看 AI 拆解出的学习点，包括词汇、表达、语法和固定搭配。
4. 勾选想要学习的内容，加入学習本。
5. 在学習本中查看和管理已保存的学习项。
6. 在復習页面进行记忆训练，标记掌握情况。
7. 学习项达到熟练状态后，进入 AI テスト。
8. 三种测试（选择、填空、判断）都通过后，该学习项进入完全掌握状态。
9. 如需使用真实 AI 解析，在設定中填写自己的 API Key、Base URL 和 Model。

## 四、设计特点

### 1. Local-first

不需要登录，不需要后端服务，不需要云同步。所有学习数据保存在浏览器本地（SQLite + localStorage）。下载 ZIP 解压后即可在本地使用，适合个人学习场景。

### 2. 用户自带 AI Provider

工具本身不绑定某个 AI 厂商，支持 OpenAI-compatible API。用户可以根据自己的情况选择 provider，例如 DeepSeek 或其他兼容服务。需要说明的是，不同 provider 的兼容性可能不同，部分 provider 可能因为 CORS 限制无法在浏览器中直接调用。

### 3. 从 AI 解析到学习闭环

普通的 AI Chat 对话，分析结果看完就消失了。这个工具的核心价值不在于聊天，而在于把 AI 解析结果转化为可复习、可测试、可毕业的学习数据。输入的句子 → AI 拆解 → 选择保存 → 复习记忆 → 多种题型测试 → 完全掌握，形成了一个完整的学习闭环。

## 五、当前 MVP 范围

**已完成：**

- AI 解析（Mock 模式 + OpenAI-compatible 真实 AI 模式）
- 学習本（查看、详情、删除、重新添加）
- 復習（间隔复习、掌握状态更新）
- AI テスト（选择题、填空题、判断题，毕业判断）
- 設定（AI Provider 切换、API Key 管理等）
- 使用方法・注意事項（首次自动弹窗 + 设置页手动入口）
- 本地持久化（刷新后数据不丢失）

**当前未包含：**

- 用户登录
- 云同步
- 后端服务
- 多设备同步
- 团队管理
- 复杂统计报表

## 六、已知限制

1. API Key 保存在浏览器本地，为非加密存储。
2. 清空浏览器数据后，学习记录可能会丢失。
3. 使用真实 AI 时，输入内容会发送给用户选择的 AI Provider。
4. 不同 OpenAI-compatible provider 的兼容性存在差异。
5. 部分 provider 可能因为 CORS 策略无法在浏览器中直接调用。
6. AI 解析结果可能不完全准确，需要用户结合实际语境判断。
7. 当前版本没有云同步和账号系统。

---

## 七、总结

Business Japanese Skill 不是一个单纯的 AI Chat 工具。它把商务日语句子解析、个人知识库、间隔复习和多种题型测试连接成一个完整的学习闭环。对正在学习商务日语的人来说，可以把工作和学习中遇到的真实表达逐步沉淀成自己的知识库。

当前版本是 MVP，重点验证核心学习流程的完整性和可用性。后续可以继续扩展 PWA 支持、数据导入导出、更多题型、更强的统计分析，以及更好的 provider 兼容性。
